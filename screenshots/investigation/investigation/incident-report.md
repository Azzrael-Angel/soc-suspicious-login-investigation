# Incident Report

## Summary
Suspicious login activity detected on employee account.

## Severity
High

## Root Cause
Likely password guessing or brute-force attack.

## Impact
Possible unauthorized access to user account.

## Actions Taken
- Reviewed Event Viewer logs
- Identified failed + successful login pattern

## Recommendations
- Reset credentials
- Enable MFA
- Block suspicious IP
- Increase monitoring
